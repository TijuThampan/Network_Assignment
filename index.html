Docker

Docker is a containerization platform that allows you to package applications and their dependencies into containers.

Ansible

Ansible is an open-source automation tool that automates software provisioning, configuration management, and application deployment.

Ubuntu

Ubuntu is a popular Linux distribution based on Debian. It's known for its ease of use and wide community support.
